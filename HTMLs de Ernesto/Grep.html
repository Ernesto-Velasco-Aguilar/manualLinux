GREP

Busca patrones en las lineas de un archivo

opciones:

  • -i ignora mayusculas
  • -v muestra lineas que NO complan con los requisitos
  • -n muestra el número de lineas junto con la salida

Ejemplo:

grep hola ./ejemplo.txt

Busca las linear que contengan "hola" en ejemplo.txt

Supongamos que tienes un archivo HTML llamado pagina.html y quieres encontrar todas las lineas qie contienen un enlace (%p). puedes usar grep de la siguiente manera:
grep "%p" pagina.html
